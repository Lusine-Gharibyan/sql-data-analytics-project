# SQL Data Analytics Project
A comprehensive collection of SQL scripts for data analytics and reporting. This repository demonstrates a variety of analytical techniques, including:
- Magnitude Analysis – Quantifying data and grouping results by dimensions like country, category, or gender.
- Ranking Analysis – Identifying top and bottom performers among products and customers.
- Trend & Time-Based Analysis – Tracking changes, growth, and running totals over time.
- Performance Analysis – Comparing current metrics to averages or previous periods.
- Segmentation Analysis – Dividing data into meaningful categories for targeted insights.
- Part-to-Whole Analysis – Evaluating differences between categories and their contribution to overall totals.

These scripts are designed to help data analysts and BI professionals quickly explore, segment, and analyze datasets within a relational database.

## Included in this repository:
- SQL scripts for analytics and reporting
- Customer and product reports with KPIs and segmentation
- Sample datasets in CSV format
- .bak file for restoring the full database in SQL Server
